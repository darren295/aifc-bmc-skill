# Trusted Business Model Canvas (`aifc-bmc`)

AI drafts your business model. You own every block.

Lay a whole business model on one page across the nine blocks of the Business Model Canvas. AI drafts each block as a confidence-labeled starting point you control — because the value is in the gaps between the blocks, not the filling.

## What it does

- Drafts all nine Business Model Canvas blocks from your real business, not textbook placeholders.
- Labels every item **evidenced (●)** or **assumption (○)** so you see exactly where to apply judgment.
- Surfaces the gaps and mismatches between blocks — the part that is actually strategy.
- Hands back an editable, self-contained canvas you keep and control.
- Always ends in a next action, never a static picture.

## Use it when

- You are mapping or rethinking a business model — a whole company, one product line, or a proposed venture.
- You are pressure-testing a pivot, a new segment, or a new revenue model.
- Someone asks "how does this business actually work?" or "where does the money come from?" and you want a structured first draft to argue with.

## Install

This repository is a [Claude Code plugin](https://code.claude.com/docs/en/plugins). Add it as a marketplace source, then install:

```shell
/plugin marketplace add darren295/aifc-bmc-skill
/plugin install aifc-bmc@aifc-bmc-skill
```

Once installed, run `/aifc-bmc` (or just ask for a business model canvas) to use it.
